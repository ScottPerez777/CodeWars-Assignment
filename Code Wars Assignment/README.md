# CodeWars Solutions

This README covers four simple Python solutions to common CodeWars problems.

## 1. Even or Odd
This solution checks if a given number is even or odd and returns `"Even"` or `"Odd"` accordingly.

## 2. Convert a Number to a String
This solution converts an integer into its string representation.

## 3. Remove String Spaces
This solution removes all spaces from the input string, returning the string with no spaces at all.

## 4. Vowel Count
This solution counts how many vowels (`a, e, i, o, u`) appear in a given lowercase string.
